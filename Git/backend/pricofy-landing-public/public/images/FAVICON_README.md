# Favicon

El favicon está configurado para usar el logo de Pricofy.

## Configuración Actual

El favicon está configurado en `app/layout.tsx` usando el logo existente:
- `/images/logo_sin_Fondo.PNG`

## Para Optimizar el Favicon

Si quieres crear versiones optimizadas del favicon, puedes:

1. Crear un archivo `icon.png` o `icon.ico` en la carpeta `app/`
2. Next.js 14 detectará automáticamente estos archivos
3. Tamaños recomendados:
   - `icon.png`: 32x32 o 64x64 píxeles
   - `icon.ico`: formato ICO estándar
   - `apple-icon.png`: 180x180 píxeles para dispositivos Apple

El favicon actual se mostrará en la pestaña del navegador usando el logo de Pricofy.
