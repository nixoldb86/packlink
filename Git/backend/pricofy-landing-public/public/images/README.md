# Directorio de Imágenes

Este directorio contiene las imágenes utilizadas en la aplicación.

## Logo

Coloca el archivo del logo de Pricofy en este directorio con el nombre `logo.png` (o `logo.svg`, `logo.jpg` según el formato que tengas).

El logo debe ser la imagen completa que incluye:
- El icono circular verde con la etiqueta de precio y checkmark
- El texto "pricofy" en minúsculas

Si el logo tiene un formato diferente, actualiza la referencia en `components/Navbar.tsx` para usar el nombre correcto del archivo.
